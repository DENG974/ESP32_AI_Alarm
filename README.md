# AI智能闹钟

基于 ESP32 的语音控制智能闹钟，支持自然语言交互，配套微信小程序可视化控制。

---

## 项目演示

| 语音指令 | 系统响应 |
|---------|---------|
| “你好” | “你好！我是AI闹钟” |
| “几点了” | “当前时间 14:30” |
| “5分钟后提醒我” | “请问做什么？” → “写作业” → “闹钟已设置：写作业” |
| “下午3点开会” | “闹钟已设置：开会” |
| “查看闹钟” | 显示所有闹钟列表 |

---

## 核心功能

- ✅ **语音识别**：百度 API 实时转文字
- ✅ **语义理解**：支持“5分钟后”、“下午3点”等自然语言
- ✅ **闹钟管理**：添加、命名、删除、持久化存储
- ✅ **时间同步**：NTP 自动校时
- ✅ **微信小程序**：可视化查看时间、闹钟列表、识别内容
- ✅ **实时反馈**：所有对话内容实时显示在小程序上
