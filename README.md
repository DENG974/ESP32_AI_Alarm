# ESP32 AI 语音智能闹钟

基于 ESP32 的端侧语音交互系统，实现自然语言控制的智能闹钟。

## 项目背景

传统闹钟交互方式单一，本项目通过语音识别技术，让用户可以用自然语言设置闹钟、查询时间，提供更自然的交互体验。

## 核心功能

- **语音识别**：集成百度 ASR API，实时将语音转为文本
- **语义解析**：支持相对时间（“5分钟后”）和绝对时间（“下午3点提醒我”）
- **时间同步**：NTP 网络时间同步，保证闹钟准确
- **多任务架构**：FreeRTOS 实现语音采集、闹钟轮询、网络通信并发

## 技术栈

| 层级 | 技术 |
|------|------|
| 硬件 | ESP32、INMP441（I2S麦克风）、MAX98357（音频放大） |
| 系统 | FreeRTOS、ESP-IDF v5.2 |
| 网络 | WiFi、NTP、HTTP/HTTPS |
| 云服务 | 百度智能云 ASR |
| 存储 | NVS（闹钟持久化） |

## 硬件接线

| 麦克风 INMP441 | 音频放大 MAX98357 |
|----------------|-------------------|
| VDD→3.3V | VIN→5V |
| WS→GPIO25 | BCLK→GPIO26 |
| SCK→GPIO33 | LRC→GPIO27 |
| SD→GPIO32 | DIN→GPIO22 |

## 语音命令示例

| 命令 | 响应 |
|------|------|
| “几点了” | 播报当前时间 |
| “5分钟后提醒我” | 设置5分钟后闹钟 |
| “下午3点提醒我开会” | 设置15:00闹钟 |

## 项目亮点

1. **内存优化**：解决 Base64 编码时内存不足问题，峰值内存从 85KB 降至 32KB
2. **并发安全**：使用互斥锁保护闹钟数据，避免 FreeRTOS 任务竞争
3. **时间同步**：NTP 自动校时，断电不丢失
